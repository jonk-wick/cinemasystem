项目概述

本项目（cinemasystem）是综合性影院管理系统，整合票务、影厅、影片等核心业务，面向影院管理员与观影用户，提升运营效率、优化观影体验，为经营决策提供数据支撑。



核心功能模块

① 影片管理模块：管理影片全生命周期，支持录入影片信息、设置排期、更新状态，提供搜索筛选功能。

② 票务管理模块：覆盖用户购票全流程，支持选座、下单、支付，同时支持管理员管理订单、退票改签。

③ 影厅管理模块：管理影厅信息（座位、设备），调度影厅使用，避免场次冲突，保障放映有序。

④ 会员服务模块：支持会员注册、积分累计，提供会员折扣、专属权益，提升用户粘性。

⑤ 数据统计模块：统计票房、观影人数、影片热度等数据，生成报表，辅助影院经营决策。
