# 校园电影购票系统
本项目旨在开发一款适配校园场景的电影购票系统，方便师生在线选座购票、查看影片、管理场次。

技术栈：前端使用Vue3，后端使用SpringBoot，数据库采用MySQL，接口文档使用Swagger

功能介绍：
① 影片管理：影片信息展示、播放时间、海报、详情管理
② 票务管理：在线选座、购票、退票、订单查看
③ 影厅管理：座位布局、场次安排、影厅信息维护
④ 会员管理：注册、登录、积分、优惠
⑤ 数据统计：票房统计、热门影片、用户分析
=======


\## 技术栈



本项目采用前后端分离架构，主要技术选型如下：



| 层级       | 技术                           |

|-----------|--------------------------------|

| 前端       | Vue 3, Vue Router, Pinia, Vite |

| UI组件库   | Element Plus                   |

| 后端       | Spring Boot 2.7, Spring Security|

| 数据库     | MySQL 8.0, Redis (缓存)        |

| ORM框架    | MyBatis-Plus                   |

| 接口文档   | Swagger 3 (Knife4j)            |

| 版本控制   | Git, GitHub                    |

| 项目管理   | Maven                          |
